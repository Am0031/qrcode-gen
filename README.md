# Deploying updates
This repo is deployed via GitHub Pages directly from the `main` branch — there's no build step. To publish a change:

```
git add .
git commit -m "your message here"
git push
```

GitHub Pages will automatically rebuild and the live site updates within a minute or two. No extra deploy command needed.
